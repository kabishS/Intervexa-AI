 🚀 Intervexa AI
Intervexa AI is a modern interview preparation platform designed to help students and job seekers improve their aptitude, coding, and problem-solving skills through interactive assessments, coding challenges, and smart study reminders.

--------------------------------------------------------------------------------------------------------

 ✨ Features

 🔐 User Login & Registration System
 🧠 Interactive Aptitude Tests
 ⏱️ Timed Questions with Instant Feedback
 💻 Coding Challenge Workspace
 📊 Multiple Difficulty Levels
 💡 Hints and Solution Explanations
 ⏰ Smart Study Reminders
 📧 Email Notifications using EmailJS
 🎨 Modern Responsive UI Design
 🌙 Professional Dark Theme Experience

--------------------------------------------------------------------------------------------------------

 🛠️ Technologies Used

 HTML5 – Page Structure
 CSS3 – Styling and Responsive Design
 JavaScript (ES6) – Application Logic
 EmailJS – Email Reminder Service
 Google Fonts – Typography
 Claude AI – UI/UX Design Assistance and Development Support

--------------------------------------------------------------------------------------------------------

 📂 Project Structure

```text
Intervexa-AI/
│
├── index.html       # Login & Registration
├── home.html        # Landing Page
├── aptitude.html    # Aptitude Test Module
├── coding.html      # Coding Challenge Module
└── reminder.html    # Reminder Management System
```

--------------------------------------------------------------------------------------------------------

 🎯 Modules

 🔐 Authentication

Provides user login and registration functionality with validation and password strength checking.

 🧠 Aptitude Test

Allows users to practice aptitude questions with timers, explanations, and score tracking.

 💻 Coding Challenges

Interactive coding environment where users can solve programming problems and test their solutions.

 ⏰ Reminder System

Helps users schedule study sessions and interview preparation reminders with email notifications.

 🏠 Landing Page

Displays platform information, features, and navigation to different modules.
